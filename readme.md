```

```

* Instalar dependencias npm

```
    $ npm install
```

* Instalar recursos

```
    $ ionic cordova resources
```

* Instalar plataforma

```
    $ ionic cordova platform add android
```

* Instalar Plugins (si es necesario)

```

```
